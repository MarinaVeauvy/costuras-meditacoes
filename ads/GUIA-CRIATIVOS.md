# Criativos Meta Ads — Raio-X Sistêmico Financeiro

## Como usar os arquivos HTML

1. Abra cada arquivo `.html` no navegador (Chrome)
2. Pressione `F12` → aba "Console" → cole o comando abaixo para screenshot perfeito:

### Para Feed (1080x1080):
```
Ctrl+Shift+M → digitar 1080x1080 → screenshot
```

### Método mais fácil:
1. Abra o HTML no Chrome
2. `Ctrl+Shift+I` (DevTools)
3. `Ctrl+Shift+M` (modo responsive)
4. Digite `1080` x `1080` (ou `1080` x `1350` para 4:5, ou `1080` x `1920` para stories)
5. Clique nos 3 pontinhos `⋮` → "Capture screenshot"
6. Salva PNG perfeito com as dimensões exatas
